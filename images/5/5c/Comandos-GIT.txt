Sequencia de comandos a serem executados no git:

1 - git checkout master (Vai para a branch master)
2 - git pull origin master (Atualiza todo o código na Branch master)
3 - git branch (Lista todas as Branches e marca com * a Branch que voce esta atualmente)
4 - git checkout -B dev-yves (Cria uma Branch para comitar suas alterações preservando o código na master)

OBS: Todos os commits deverao ser realizados na sua branch e para subir um código para o github basta usar o comando: git push origin dev-yves (Não usar git push origin master)
